# Nothing Landing-Page

Hey there!
Welcome to **Nothing Landing-Page**, my **responsive landing page** project. It’s a clean and modern one-page layout built with **HTML** and **SCSS (BEM)**, using **CSS Grid** and **Flexbox** for pixel-perfect responsiveness across all devices — desktop, tablet, and mobile.

## Technologies Used
- **HTML5** — semantic and structured markup
- **SCSS (BEM)** — clean, reusable and modular styles
- **CSS Grid** — for the main layout
- **Flexbox** — for flexible alignment
- **Responsive design** — fully adapted for all screen sizes
- **Burger menu** — smooth navigation between sections
